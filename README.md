# DevOps Lab 2 Zadanie 2

Projekt demonstracyjny z automatyzacją GitHub Actions.

## Opis
Prosta aplikacja z automatycznym testowaniem i budowaniem przy użyciu GitHub Actions.

## Struktura
- `src/` - kod źródłowy aplikacji
- `tests/` - testy automatyczne
- `docs/` - dokumentacja
- `.github/workflows/` - definicje workflow GitHub Actions
